# MDA_MST
MST's code for our TCS (Thermal Control System) unit on the FlatSat


It is important to note that additional software not mentionned in the code are neceessary:


- Virtual Serial Port Tools: Used for splitting physical COM ports into virtual ones
- Environemental Tester: The Software that comes with the Digital Data logger
            => It allows reading Temp , but aonly sandboxed in their app and only allows exporting it into a CSV afterwards.

- Arduino IDE (or another IDE) for C code for ESP32.
